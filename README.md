## Instructions for install dependencies ðŸ“–

#### Installing NPM 

Run the following command to install NPM on Linux distributions:

    $ sudo apt install npm

#### Installing Node

Run the following command to install NPM on Linux distributions:

    $ sudo apt install nodejs

#### Installing Angular 

Run the following command to install NPM on Linux distributions:

    $ npm install -g @angular/cli

#### Installing MongoDB

Run the following command to install NPM on Linux distributions:

    $ sudo apt install -y mongodb
